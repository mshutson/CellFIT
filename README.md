# CellFIT
This repository contains Mathematica .nb files that implement CellFIT algorithms for inferring cellular tensions and pressures from images of epithelial cells. CellFIT algorithms are as described in the following two papers:

G.W. Brodland, J.H. Veldhuis, S. Kim, M. Perrone, D. Mashburn and M.S. Hutson (2014) "CellFIT: a cellular force-inference toolkit using curvilinear cell boundaries" PLoS ONE 9: e99116 (15pp), https://doi.org/10.1371/journal.pone.0099116.

J.H. Veldhuis, D. Mashburn, M.S. Hutson, G.W. Brodland "Practical Aspects of the Cellular Force Inference Toolkit (CellFIT)", in Methods in Cell Biology Volume 125: Biophysical Methods in Cell Biology, edited by E. Paluch, Chapter 18, pp. 331-351 (Elsevier, 2015), https://doi.org/10.1016/bs.mcb.2014.10.010. 

Modifications of the segmentation and tangent vector steps were implemented to ensure that CellFIT results were invariant to image rotations and reflections. These steps were taken for the following article:

I. Goodspeed, Z. Li, M. Shane Hutson, M.A. Chrenek, J.H. Boatright, J.M. Nickerson, Y. Jiang (2026) "Stress Inference in Retinal Pigment Epithelium in C57BL/6J Mouse" <i>Submitted</i>.
